Simple Widget plugin to use in your wordpress installation, simply enter websites url (can be subpage) and it will display.

== Description ==

Uses the same URL for both pagerank and alexa ranking.

== Installation ==


1. Upload `vdwp-alexa.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Widget options and drag the alexa widget to your desired menu
